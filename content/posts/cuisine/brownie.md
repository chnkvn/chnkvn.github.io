## Infos {#infos}

| Quantité    | Préparation | Cuisson |
|-------------|-------------|---------|
| 6 personnes | 0:10        | 0:20    |


## Ingrédients {#ingrédients}

| Ingrédient              | Quantité | Remarques                                    |
|-------------------------|----------|----------------------------------------------|
| Chocolat noir pâtissier | 200g     |                                              |
| Beurre demi-sel         | 50g      | Possible d'utiliser de la margarine          |
| Lait                    | 180g     | Possible d'utiliser une alternative végétale |
| Sucre                   | 90g      |                                              |
| Poudre d'amande         | 100g     |                                              |
| Fécule                  | 50g      | E.g. Maïzena                                 |
| Cacao en poudre         | 10g      |                                              |
| Autres fruits secs      | 100g     | J'utilise principalement de la noisette      |


## Instructions {#instructions}

1.  Préchauffer le four à 180°C
2.  Fondre le chocolat, le beurre et le lait et mélanger
3.  Mélanger les poudres ensemble
4.  Concasser les fruits secs
5.  Mélanger tous les ingrédients ensemble, puis transvaser dans un moule recouvert de papier sulfurisé
6.  Enfourner 25 minutes
7.  Laisser refroidir à température ambiante avant de démouler et de couper pour que le brownie se raffermisse.

Bon appétit !
